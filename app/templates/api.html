API Documentation

Get list of all athletes in the database with their parameters:

Example:

/api/athletes

Get athlete's best results:

Parameters:

  • athleteid - first 3 letters of surname + first 3 letters of name + year of birth + gender (F or M)

Example:

/api/athletes?athlete_id=REBIGO1989M

Get athlete's results for particular distance:

Parameters:

  • athleteid - first 3 letters of surname + first 3 letters of name + year of birth + gender (F or M)
  • distance - meters + stroke (FREE, BREAST, BACK, FLY, MEDLEY)

Example:

/api/athletes_results?athlete_id=REBIGO1989M&distance=100 FLY

Get list of all meets in the database:

Example:

/api/meets

Get results of particular event in the selected meet:

Parameters:

  • meet_code - shortcut of the meet
  • distance
  • event_gender

Example:

/api/meets?meet_code=LMP_WAW_2023&event_gender=M&distance=100 FLY

Get top results of particular event:

Parameters:

  • year - parameter not obligatory, if omitted rankings will be gotten for all year
  • course - LCM or SCM
  • event_gender
  • distance
  • records_nb - number of records to be listed (not obligatory)

Example:

/api/rankings?year=2023&event_gender=M&distance=100 FLY&course=LCM&records_nb=5

Get national records for particular event for all age groups:

Parameters:

  • event_gender
  • distance

Example:

/api/records?event_gender=M&distance=100 FLY
